# Politica cookie

## 1. Definizione
Un cookie è un piccolo file depositato sul tuo dispositivo durante la visita dell'applicazione.

## 2. Cookie utilizzati
- Sessione : strettamente necessari al funzionamento (autenticazione).
- Misurazione di audience : [anonimizzata se del caso].

## 3. Gestione
Puoi gestire i cookie tramite le impostazioni del tuo browser.
